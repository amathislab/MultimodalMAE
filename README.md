# MultimodalMAE

Package for action recognition with multimodal masked autoencoders. This package enables behavioral analysis from video, audio, depth cameras, etc. 

The work has been road tested and developed for three recent projects from the lab:
- [MammalAlps](https://eceo-epfl.github.io/MammAlps/) (CVPR highlight 2025)
- [EPFL smart kitchen](https://github.com/amathislab/EPFL-Smart-Kitchen) (NeurIPS 2025)
- [DLC2action](https://github.com/amathislab/DLC2action) (biorxiv 2025)

It will appear in July 2026 -- we apologize for the delay!!
